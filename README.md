# MicroTools 🛠️

Free online developer tools. Simple, fast, useful.

## What Is This?

A collection of small, handy web tools for developers - all in one file.

## Tools Included

- **Lorem Ipsum Generator** - Generate placeholder text
- **Password Generator** - Create secure random passwords
- **JSON Formatter** - Validate and pretty-print JSON
- **Base64 Encoder/Decoder** - Encode and decode Base64
- **UUID Generator** - Generate random UUIDs
- **Unix Timestamp Converter** - Convert timestamps to dates

## How to Use

1. Clone this repository:
   ```
   git clone https://github.com/YOUR_USERNAME/microtools.git
   ```

2. Open `index.html` in any web browser

3. That's it! No installation or server needed.
